# AI-Based Health Assistant

This project provides:
- 773 general disease predictions (text-based)
- Skin cancer detection (image-based)

# Setup
1. Install Python dependencies:cd backend,run virtual\scripts\activate 
   then run  pip install -r requirements.txt
2. Run Django server: python manage.py runserver
3. create new terminal and run run cd frontend\myapp 
4. then run npm run dev 
5. copy the displayed server address and past on the browser,that is it
